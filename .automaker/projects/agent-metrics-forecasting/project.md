# Project: Agent Metrics & Forecasting

## Goal
Comprehensive timing, cost, and capacity tracking for AI agents. Capture the full lifecycle (backlog to merge), per-execution records with token usage, status transition history, and capacity snapshots. Build metrics aggregation and project forecasting on top of the raw data.

## Milestones
1. Feature Lifecycle Timestamps - Foundation layer: add persistent timing primitives to Feature type.
2. Execution Records - Per-execution records for every agent run with timing, cost, tokens, model.
3. Metrics Service & API - Aggregation layer: throughput, cycle time, cost rates, capacity, forecasting.
