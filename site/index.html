01 / Describe

Signal Intake

Ideas enter from anywhere — a Discord message, a Linear issue, a conversation. The system classifies the signal and routes it to the right pipeline. No context is lost, no idea falls through the cracks.

03 / Plan

Decomposition

Approved projects decompose into milestones, phases, and features — each with acceptance criteria, complexity estimates, and dependency graphs. The system knows what to build and in what order.

04 / Build

Autonomous Execution

AI agents implement features in isolated git worktrees. Auto-mode orchestrates concurrency, respects dependencies, and escalates model complexity when needed. Each agent works in its own branch — the main codebase stays clean.

05 / Ship

Quality Pipeline

Four layers of quality guardrails: TypeScript strict mode, automated formatting, CI test suites, and AI code review. PRs auto-merge when all checks pass. No human bottleneck in the merge path.

06 / Learn

Reflection Loop

Every completed project triggers retrospectives, metrics analysis, and improvement tickets. The system upgrades itself through the same pipeline it uses for your work. Lessons compound. Costs drop. Quality rises.
