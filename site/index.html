Features

Everything you need to run an autonomous dev team

protoLabs Studio gives you a Kanban board, AI agent orchestration, and a full CI-integrated pipeline — all running locally on your machine.

Kanban Board

Visual feature management with drag-and-drop. Track every feature from backlog through review to done. Epics, dependencies, and priority ordering built in.

AI Agent Teams

Tiered model selection — Haiku for quick tasks, Sonnet for standard work, Opus for architecture. Agents auto-escalate on failure.

Auto-Mode

Queue features and let the system work through them autonomously. Respects dependency order, manages concurrency, and creates PRs automatically.

Git Worktree Isolation

Every feature runs in its own git worktree. Agents never touch your main branch. Parallel development with zero conflicts.

Linear + Discord Integration

Pipe features in from Linear issues or Discord messages. Signals get classified, triaged, and routed to the right pipeline automatically.

Quality Pipeline

CI checks, AI code review, automated formatting, and TypeScript strict mode. PRs auto-merge when all checks pass. No bottlenecks.

How It Works

Describe what you want. Watch it ship.
